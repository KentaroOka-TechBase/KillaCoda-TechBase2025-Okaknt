# Step 4: 環境変数が設定されない謎

どうやら環境変数は設定されているようですが、一番上の文字列のみ、
```bash
Client-side (CLIENT_MESSAGE): この演習の名前は(not available)です
```
と表示されてしまい、値がうまく反映されていないようです。
インターネット検索などを用いていいので、この環境変数だけが反映されていない理由を調べてみてください。

理由がわかったら、エディタで.env.localファイルやEnvViewer.tsxを修正し、一番上の文字列が以下のように表示され続けることを確認しましょう。<br>
これらのファイルを保存すると、数秒後に画面に変更が反映されます。<br>

```bash
Client-side (CLIENT_MESSAGE): この演習の名前はTechBase2025_CLIENTです
```
※注意
- 何も設定しなくても、一瞬だけ上記のように表示されることもありますが、これが表示され続けることを目指してください。

- 環境変数名や、EnvViewer.tsxで使う変数名は自由に設定していただいて構いませんが、「この演習の名前はTechBase2025_CLIENTです」と表示できるようにしてください。

- ファイルを保存すると自動的に変更がサーバに反映されるため、サーバの再起動は不要です。


上記達成できた場合や、回答を知りたい場合は「NEXT」のボタンを押下してください。
