# Step 3: 表示される文字列の確認

ここに表示される文字列は、<br>
```bash
/next-env-demo
```
配下の **.env.local**フォルダで設定している環境変数を用い、<br>
```bash
/next-env-demo/src/components/EnvViewer.tsx
```
によって環境変数を読みだして表示されています。

実際にコードや値を確認してみましょう。<br>
ターミナルにてコマンドを使うか、<br>
画面中央上部付近の Editor ボタンを押下してエディターを開くなどして、<br>
**/next-env-demo/src/components/EnvViewer.tsx**　や<br>
**/next-env-demo/.env.local**の値を確認してみてください。

エディターを使う場合、画面は以下のようになります。
![Scan results](assets/editorimage.png)
値を確認したら、「NEXT」のボタンを押下してください。
